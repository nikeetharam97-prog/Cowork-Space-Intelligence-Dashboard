# Cowork-Space-Intelligence-Dashboard

This project combines Interior Design and Data Analytics to evaluate the effectiveness of a coworking space through quantitative analysis.

Using a real-world workspace design consisting of 15 zones across 9,305 sq.ft., the project analyzes space utilization, occupancy efficiency, cost allocation, flexibility, adjacency relationships, and overall zone performance.

Business Problem:
Traditional workplace design decisions are often based on experience and qualitative judgment. This project demonstrates how data analytics can be used to measure and optimize workspace performance using objective metrics.

Key Findings
1. Space Allocation
Wellness-focused spaces occupy only 7.2% of the total floor area, below the recommended industry benchmark of 12–15%.

2. Space Efficiency
Higher space allocation per person does not guarantee better performance. Compact, flexible, and naturally lit spaces consistently outperformed larger zones.

3. Cost Effectiveness
Open workstations generated the best value proposition at approximately ₹4,000 per person, while cabins showed higher costs with lower flexibility.

4. Design Strategy
The strongest-performing zones shared two characteristics:

--High Adjacency Score
--High Flexibility Score

This pattern provides a replicable design framework for future workspace expansion.

Tools Used
Python.Pandas.Matplotlib

The project demonstrates how workplace design decisions can be supported by data, helping organizations improve utilization, reduce costs, and create more effective work environments.
